IRP_MJ_DEVICE_CONTROL is IRP_Dispatch_Routine. IRP_MJ_DEVICE_CONTROL is _IO_STACK_LOCATION IOCTL dispatch routine defined in the follow structre.

NTSTATES Dispatch DeviceControl {
  __in struct _DEVICE_OBJECT *DeviceObject,
  __in struct _IRP *Irp
}

_IRP is structure of IRP(I/O request packet) structures pointer.
