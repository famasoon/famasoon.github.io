Executing functions in Rust with winapi

June 4, 2023 · 2 min · 388 words · FAMASoon | Suggest Changes

Motivation#

I want to how to allocate memory on windows and execute code in buffers.

winapi#

winapi provides raw FFI bindings to all of Windows API. So, I will use this crate.

code#

When run command cargo run, popup calc.exe
